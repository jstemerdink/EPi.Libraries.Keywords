Add the [KeywordsMetaTag] to your keywords property.

Add an Alchemy API key to your appsettings <add key="seo.alchemy.key" value="YourKey" />

Or add a KeywordGenerationSettingsBlock to your start page type, if you would like to configure the settings that way. 

Add the [KeywordGenerationSettings] attribute to the Block.

Note that not all languages are supported by Alchemy (http://www.alchemyapi.com/)
